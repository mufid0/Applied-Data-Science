# 🚀 Data Science Projects

## 📌 Overview

This repository showcases applied data science projects focused on solving real-world problems using structured workflows and machine learning techniques.

Each project demonstrates the ability to transform raw data into actionable insights and build predictive models using industry-standard practices.

---

## 🧠 Approach

All projects follow a consistent and reproducible pipeline:

1. **Problem Definition**
   Understanding the business context and defining objectives

2. **Data Exploration (EDA)**
   Analyzing patterns, distributions, and relationships

3. **Data Cleaning & Preparation**
   Handling missing values and ensuring data quality

4. **Feature Engineering**
   Creating meaningful features to improve model performance

5. **Model Development**
   Training and comparing machine learning models

6. **Evaluation & Optimization**
   Using appropriate metrics and tuning techniques

7. **Insights & Conclusions**
   Translating results into actionable insights

A complete end-to-end machine learning project focused on predicting customer churn and identifying key factors influencing customer retention.

### 🚢 Titanic Survival Prediction

A classification project based on the Titanic: Machine Learning from Disaster, focused on predicting passenger survival using structured data.

**Key Highlights:**

* Data cleaning and handling missing values
* Feature engineering (Title extraction, FamilySize, IsAlone, binning)
* Categorical encoding and preprocessing
* Model training using XGBoost
* Kaggle-ready submission pipeline
